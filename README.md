# escape-from-dungeon
1. 2D, top-down, maze adventure

2. Моя игра будет представлять из себя небольшую 2D-игру с видом сверху. Основной концепцией проекта станет прохождение лабиринта, в котором игроку необходимо исследовать подземелье, избегать столкновений с врагом, найти ключ и добраться до выхода.
Игровой процесс будет ориентирован на реакцию, внимательность и навигацию по карте. Игрок сможет управлять персонажем с помощью клавиатуры, перемещаясь по игровому полю и взаимодействуя с объектами окружения. На уровне будут расположены стены, ограничивающие передвижение, ключ для открытия двери и противник, представляющий угрозу для игрока.
Разработка проекта позволит изучить основы объектно-ориентированного программирования, работу с игровыми циклами, обработкой событий, графикой, анимацией и системой контроля версий Git.

3. Язык программирования - Python.

4. Среда разработки - PyCharm.

5. Типы файлов: png, mp3.
